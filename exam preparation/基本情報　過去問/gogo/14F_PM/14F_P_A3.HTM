問8
　
設問　 ＜解答＞　a　イ　　　b　ウ　　　c　キ　　　d　ウ

＜解説＞空欄aについて
クラスGATEでWire型のオブジェクト配列inputが宣言されているので，空欄aでWire型配列inputの生成が必要となる。次のfor文で要素数nInputsのインスタンスオブジェクトinput [ i ]を生成している。したがって空欄aで，要素数nInputsのWire型配列inputの生成が必要となり，空欄aはnew Wire [ nInputs ]に該当する。

空欄bについて
空欄bは，出力信号線outputを指定されたゲートotherGateのnthInput番目の入力信号線input [ nthInput ]に接続するメソッドconnectOutputToにおける処理である。したがってotherGateのnthInput番目の入力信号線input [ nthInput ]に，メソッドsetValueにより出力信号線outputの論理値output.getValue ( )をセットすればよく，空欄bはotherGate. input [ nthInput ]. setValue ( output.getValue ( ) )となる。

空欄cについて
NotGateクラスのメソッドtick ( )は否定論理演算を行なうメソッドで，入力信号線のboolean型の論理値の否定を出力信号線outputにセットして返せばよい。したがって空欄cは！input [ 0 ].getValue ( )に該当する。

空欄dについて
AndGateクラスのメソッドtick ( )はAND演算を行なうメソッドで，2つの入力信号線のboolean型の論理値のAND演算の結果を出力信号線outputにセットして返せばよい。したがって空欄dはinput [ 0 ].getValue ( ) && input [ 1 ].getValue ( )に該当する。
